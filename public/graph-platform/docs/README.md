## 📖 项目简介

**图论科研平台 Pro v6.1 (Smart UI)** 是一个基于 Web 的纯前端图论可视化与计算平台，专为图论研究人员、教师和学生设计。平台提供强大的交互式图编辑能力、15 种经典图生成器、图指标计算，以及 9 种专业的可视化高亮模式。v6.1 版本引入了全新的动态网格系统和优化的交互体验。

### 🎯 核心定位

- **科研工具** - 快速验证图论猜想、构造反例、参数计算
- **教学平台** - 图论课程演示、指标可视化教学、作业辅助
- **学习工具** - 理解图论概念、算法实验、可视化辅助学习

### ✨ 项目亮点

- ✅ **纯前端实现** - 无需服务器，可离线运行，即开即用
- ✅ **零依赖安装** - 直接在浏览器中打开 `index.html` 即可使用
- ✅ **高性能计算** - Web Worker 异步处理，支持密集型图算法
- ✅ **专业导出** - 支持高清 PNG 导出及 LaTeX (TikZ) 代码生成
- ✅ **现代交互** - 支持无限画布、鼠标/空格平移、智能吸附、物理布局
- ✅ **丰富内容** - 15 种图生成器，图指标计算，9 种可视化模式

## 🚀 快速开始

### 方式一：本地打开（推荐）

1. **下载项目**
   ```bash
   git clone <repository-url>
   cd graph-platform
   ```

2. **打开应用**
   由于项目使用了 ES Modules 和 Web Worker，建议通过本地服务器运行：
   - **Windows**: 双击 `start_server.bat` (如果安装了 Node.js/Python)
   - **Python**: `python -m http.server 8080`
   - **Node**: `npx http-server`

   访问: `http://localhost:8080`

### 5 分钟上手

1. **创建节点** - 双击画布空白处
2. **创建边** - 按住 `Shift` + 从节点拖拽到另一节点
3. **移动视图** - 按住 `空格键` + 左键拖拽，或使用鼠标中键
4. **选择对象** - 单击选中，`Ctrl` + 点击多选，或直接框选
5. **生成图** - 点击左侧 ✨ 图标，选择图类型（如"完全图 K₅"）
6. **算法分析** - 右侧面板自动显示图的各项参数（染色数、直径等）
7. **可视化** - 点击参数旁的 👁️ 眼睛图标查看指标高亮
8. **导出** - 点击导出按钮，保存为图片或 LaTeX 代码

### 2. 15 种经典图生成器

支持生成多种标准图结构，用于快速实验：
- **基础图**: 完全图 $K_n$, 圈图 $C_n$, 路图 $P_n$
- **特殊图**: 彼得森图, 格勒奇图, 希伍德图, 轮图 $W_n$
- **结构图**: 二维网格, 超立方体 $Q_n$, 完全二部图 $K_{n,m}$
- **树与随机**: 满二叉树, 星图, Erdős-Rényi 随机图
- **极值图**: 图兰图 $T_{r,n}$

### 3. 图指标计算

自动计算并实时更新以下参数：
- **基础**: $|V|$, $|E|$, $\Delta$, $\delta$, 平均度
- **结构**: 直径 (diam)
- **染色**: 顶点染色数 $\chi(G)$, 边染色数 $\chi'(G)$
- **连通性**: 顶点连通度 $\kappa$, 边连通度 $\lambda$ (最大流/最小割)
- **高级**: 最大独立集 $\alpha(G)$, 哈密顿路径/圈, 两点间距离
- **代数**: 邻接矩阵/拉普拉斯矩阵及其特征值谱

### 4. 导出与分享

- **PNG 导出**: 支持 1x/2x/4x 分辨率，可选透明/白色/深色背景。
- **TikZ 导出**: 生成高质量 LaTeX 代码，直接用于学术论文。
- **JSON 保存**: 保存当前图结构，随时恢复工作。

## 🛠️ 技术栈

### 核心技术

- **Frontend**: Vanilla JavaScript (ES6+), HTML5 Canvas API
- **Computing**: Web Workers (后台多线程计算)
- **Math**: Math.js (矩阵运算), KaTeX (公式渲染)
- **Styles**: CSS Variables (深色/浅色主题), Font Awesome (图标)

### 架构亮点

- **动态渲染**: 基于 Canvas 的高性能渲染引擎，支持自定义动态网格。
- **非阻塞计算**: 耗时算法（如连通度、哈密顿圈）在 Worker 中运行。
- **模块化设计**: 分离渲染(Renderer)、物理(Physics)、算法(Algorithms)等模块。
