# Graph Platform Pro - Quick Reference

## Basic Operations

| Action | Method |
|------|------|
| Create node | Double-click on empty canvas |
| Create edge | `Shift` drag from one node to another |
| Create and connect new node | `Shift` drag from node to empty space |
| Move nodes | Drag selected nodes |
| Box select | Drag on empty space |
| Pan view | Space + left drag / middle mouse drag |
| Zoom view | Mouse wheel |
| Delete elements | Select then press `Delete` / `Backspace` |

---

## Common Shortcuts

| Shortcut | Action |
|--------|------|
| `Ctrl + Z` | Undo |
| `Ctrl + Y` / `Ctrl + Shift + Z` | Redo |
| `Ctrl + S` | Save (JSON) |
| `Ctrl + O` | Open (JSON) |
| `Ctrl + A` | Select all |
| `Ctrl + C / X / V` | Copy / Cut / Paste |
| `Ctrl + D` | Duplicate |
| `0` | Reset view |
| `Escape` | Clear selection / Close modal |

---

## Graph Metrics & Visualization

- |V|, |E|, maximum degree Δ, minimum degree δ, average degree d
- Diameter diam(G)
- Graph coloring χ(G), edge coloring χ'(G)
- Maximum independent set α(G)
- Distance dist(u, v) (requires two selected nodes)
- Vertex connectivity κ / articulation points (|V| ≤ 50)
- Edge connectivity λ / bridges (cut edges) (|V| ≤ 50)
- Hamiltonian path / cycle (|V| ≤ 24)
- Adjacency matrix A, Laplacian matrix L, and eigenvalues
