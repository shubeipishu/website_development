# 图论科研平台 Pro - 快速参考卡

## 基本操作

| 操作 | 方法 |
|------|------|
| 创建节点 | 双击画布空白处 |
| 创建边 | `Shift` 从节点拖拽到另一个节点 |
| 创建并连线新节点 | `Shift` 从节点拖拽到空白处 |
| 移动节点 | 拖拽已选节点 |
| 框选 | 拖拽空白处 |
| 平移视图 | 空格 + 左键拖拽 / 中键拖拽 |
| 缩放视图 | 鼠标滚轮 |
| 删除元素 | 选中后按 `Delete` / `Backspace` |

---

## 常用快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + Z` | 撤销 |
| `Ctrl + Y` / `Ctrl + Shift + Z` | 重做 |
| `Ctrl + S` | 保存 (JSON) |
| `Ctrl + O` | 打开 (JSON) |
| `Ctrl + A` | 全选 |
| `Ctrl + C / X / V` | 复制 / 剪切 / 粘贴 |
| `Ctrl + D` | 复制副本 |
| `0` | 重置视图 |
| `Escape` | 取消选择 / 关闭弹窗 |

## 图指标与可视化

- |V|、|E|、最大度 Δ、最小度 δ、平均度 d
- 直径 diam(G)
- 图着色 χ(G)、边着色 χ'(G)
- 最大独立集 α(G)
- 两点距离 dist(u, v)（需选中两个节点）
- 点连通度 κ / 割点（|V| ≤ 50）
- 边连通度 λ / 割边（|V| ≤ 50）
- 哈密顿路 / 圈（|V| ≤ 24）
- 邻接矩阵 A、拉普拉斯矩阵 L 及特征值
