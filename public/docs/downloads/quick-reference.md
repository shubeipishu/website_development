# 图论科研平台 Pro - 快速参考卡

## 基本操作

| 操作 | 方法 |
|------|------|
| 创建节点 | 单击画布空白处 |
| 创建边 | `Shift` + 点击两个节点 |
| 移动节点 | 拖拽节点 |
| 平移视图 | 拖拽画布空白处 |
| 缩放视图 | 鼠标滚轮 |
| 删除元素 | 选中后按 `Delete` |

## 算法复杂度

| 算法 | 时间复杂度 (LaTeX) |
|------|-------------------|
| BFS | $O(V + E)$ |
| DFS | $O(V + E)$ |
| Dijkstra | $O((V+E) \log V)$ |
| Bellman-Ford | $O(VE)$ |
| Ford-Fulkerson | $O(E \cdot f^*)$ |

## 欧拉公式
对于平面连通图，有：
$$ V - E + F = 2 $$
