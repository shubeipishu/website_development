## 安装与运行

### 在线使用（推荐）

直接访问在线版本即可使用，无需安装。

### 本地运行

由于项目使用了 ES Modules 和 Web Workers，需要通过 HTTP 服务器访问。

**使用 Python:**
```bash
cd graph-platform
python -m http.server 8080
# 访问 http://localhost:8080
```

**使用 Node.js:**
```bash
npx http-server -p 8080
```

**使用 Docker:**
```bash
docker-compose up -d --build
# 访问 http://localhost
```

## 算法可视化

### 支持的算法

#### 遍历算法
- **广度优先搜索 (BFS)** - O(V + E)
- **深度优先搜索 (DFS)** - O(V + E)

#### 最短路径
- **Dijkstra 算法** - O((V + E) log V)，不支持负权边
- **Bellman-Ford 算法** - O(VE)，支持负权边

#### 网络流
- **Ford-Fulkerson 算法** - O(E × max_flow)

#### 图着色
- **贪心着色** - 使用最少颜色给节点着色

#### 连通性
- 连通分量、强连通分量、割点检测、桥边检测

### 可视化颜色说明

| 颜色 | 含义 |
|------|------|
| 🟡 黄色 | 当前正在访问 |
| 🟢 绿色 | 已完成访问 |
| 🔵 蓝色 | 在队列/栈中等待 |
| 🔴 红色 | 路径高亮 |

## 常见问题

### Q: 为什么双击 HTML 打开后功能不正常？

**A:** 浏览器对 `file://` 协议有安全限制。请使用 HTTP 服务器访问。

### Q: 如何保存我的图形？

**A:** 使用 `Ctrl + S` 保存为 JSON 文件，可以通过 `Ctrl + O` 重新加载。

### Q: 算法运行太快看不清？

**A:** 使用控制面板调节动画速度，或使用「步进」模式逐步执行。
