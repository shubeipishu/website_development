# 图指标与可视化

平台会自动计算图指标，并提供可视化高亮（不是逐步算法动画）。

## 实时计算与可视化

- 图一旦修改，会自动重新计算指标。
- 计算量较大的指标在 **Web Worker** 中异步完成，避免阻塞界面。
- 点击右侧面板参数旁的 👁️ 图标即可开启或关闭对应高亮。

## 指标列表（与面板一致）

### 基础统计

- |V|、|E|
- 最大度 Δ、最小度 δ、平均度 d

### 结构指标与高亮

- **直径 diam(G)**：高亮直径路径（非连通图显示 ∞，此时无法高亮）。
- **图着色 χ(G)**：节点着色，并显示颜色编号。
- **边着色 χ'(G)**：边着色，边中点显示颜色编号。
- **最大独立集 α(G)**：高亮属于最大独立集的节点。
- **两点距离 dist(u, v)**：仅当**选中两个节点**时显示，并高亮最短路径。

### 连通性与割

当图连通且规模不大时，会计算并高亮：

- **点连通度 κ** 与 **割点**（闪烁提示）。
- **边连通度 λ** 与 **割边**（闪烁提示）。

> 规模限制：当 |V| > 50 时，连通性与割点/割边会显示为 `>50(skip)`。

### 哈密顿路 / 圈

- |V| ≤ 24 时尝试求解并高亮路径/圈。
- |V| > 24 时显示 `>24(skip)`。

### 矩阵与谱

- 邻接矩阵 A 与拉普拉斯矩阵 L
- A 与 L 的特征值（按从大到小显示）

## 使用方法

1. 创建或导入图形
2. 在右侧面板查看指标值
3. 点击指标左侧的 👁️ 图标进行可视化高亮
4. 再次点击同一图标可退出高亮
