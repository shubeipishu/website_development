# 算法可视化

图论平台支持 15+ 种经典图算法的可视化演示。

## 支持的算法

### 遍历算法

#### 广度优先搜索 (BFS)

**时间复杂度**: O(V + E)

从指定起点开始，逐层遍历图中所有可达节点。

```
用途：
- 最短路径（无权图）
- 连通性检测
- 层次遍历
```

#### 深度优先搜索 (DFS)

**时间复杂度**: O(V + E)

沿着一条路径尽可能深入，然后回溯。

```
用途：
- 拓扑排序
- 环检测
- 连通分量
```

### 最短路径

#### Dijkstra 算法

**时间复杂度**: O((V + E) log V)

求解单源最短路径问题。

> ⚠️ **注意**: 不支持负权边

#### Bellman-Ford 算法

**时间复杂度**: O(VE)

支持负权边的单源最短路径算法。

### 网络流

#### Ford-Fulkerson 算法

**时间复杂度**: O(E × max_flow)

求解最大流问题。

### 图着色

#### 贪心着色

使用尽可能少的颜色给图的节点着色，使相邻节点颜色不同。

### 连通性

| 算法 | 功能 |
|------|------|
| 连通分量 | 查找无向图的连通分量 |
| 强连通分量 | 查找有向图的强连通分量 |
| 割点检测 | 查找关键节点 |
| 桥边检测 | 查找关键边 |

## 使用方法

1. 创建或导入图形
2. 从右侧面板选择算法
3. 设置参数（如起始节点）
4. 点击「运行」按钮
5. 使用「暂停」「步进」控制动画

## 可视化说明

动画中的颜色含义：

| 颜色 | 含义 |
|------|------|
| 🟡 黄色 | 当前正在访问 |
| 🟢 绿色 | 已完成访问 |
| 🔵 蓝色 | 在队列/栈中等待 |
| 🔴 红色 | 路径高亮 |
