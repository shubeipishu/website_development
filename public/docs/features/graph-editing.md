# 图形编辑

本文档介绍图论平台的图形编辑功能。

## 基本操作

### 创建节点

- **单击画布空白处** - 在点击位置创建新节点
- 节点会自动分配编号（从 0 开始）

### 创建边

1. 按住 `Shift` 键
2. 点击起始节点
3. 点击目标节点
4. 释放 `Shift` 键

### 选择与移动

| 操作 | 说明 |
|------|------|
| 单击节点 | 选中节点 |
| 拖拽节点 | 移动节点位置 |
| 拖拽画布 | 平移视图 |
| 滚轮缩放 | 缩放视图 |

### 删除元素

1. 选中要删除的节点或边
2. 按 `Delete` 或 `Backspace` 键

## 高级功能

### 批量操作

- `Ctrl + A` - 全选所有节点
- `Ctrl + D` - 取消选择
- 框选 - 按住并拖拽选择多个节点

### 节点属性

双击节点可以编辑：
- 节点标签
- 节点颜色
- 节点大小

### 边属性

双击边可以编辑：
- 边权重
- 边样式（实线/虚线）
- 边颜色

## 快捷键一览

| 快捷键 | 功能 |
|--------|------|
| `Shift + 点击` | 创建边 |
| `Delete` | 删除选中元素 |
| `Ctrl + Z` | 撤销 |
| `Ctrl + Y` | 重做 |
| `Ctrl + S` | 保存图形 |
| `Ctrl + O` | 打开图形 |
| `Space` | 暂停/继续动画 |
