❓ 常见问题

推荐使用最新版 Chrome、Firefox 或 Edge 浏览器。由于使用了 ES6 模块和 Web Worker，不支持 IE 浏览器。
由于项目使用了 ES Modules，需要通过 HTTP 服务器访问。最简单的方式是使用 Python: python -m http.server 8080，然后访问 http://localhost:8080/apps/graph-platform/
目前支持 15+ 种算法，包括：BFS/DFS 遍历、最短路径（Dijkstra）、最大流（Ford-Fulkerson）、图着色、连通性检测等。详见文档中心。
